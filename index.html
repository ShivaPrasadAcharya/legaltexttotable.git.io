How to add a law file

1. Create a .js file in the same folder as index.html:

window.lawEvidenceAct2031 = {
  metadata: {
    title: "My Act, 2081",  shortTitle: "My Act",
    year_bs: "२०८१",        year_ad: "2024",
    language: "Nepali",     version: "1.0",
    source: "Nepal Law Commission",
    lastUpdated: "2024-01-01",
    description: "Brief description."
  },
  content: [
    { "id": "१",   "law": "१. संक्षिप्त नाम…", "remark1": "", "remark2": "" },
    { "id": "१.१", "law": "(१) …",              "remark1": "", "remark2": "" }
  ]
};

2. Add to index.html before </body>:

<script src="my_act.js"></script>

3. Reload page → click ⊕ Law Files → your file appears as a button.

💡 Variable must start with window.law + capital letter, e.g. window.lawMyAct.
